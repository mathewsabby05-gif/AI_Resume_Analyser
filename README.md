# AI Resume Analyzer & Job Match System

## Project Overview
AI Resume Analyzer is a Streamlit-based web application that analyzes resumes and compares them with job descriptions to determine compatibility. It calculates a match score, identifies missing skills, and provides resume improvement suggestions to help users optimize their job applications.

## Features
- Upload resume in PDF format  
- Extract text automatically from resume  
- Compare resume with job description  
- Generate job match percentage  
- Identify missing skills and keywords  
- Provide resume improvement suggestions  
- ATS-friendly optimization tips  
- Clean and interactive Streamlit UI  

## Tech Stack
- Python  
- Streamlit  
- Pandas  
- PyPDF2  
- Scikit-learn  
- TF-IDF Vectorizer  
- Cosine Similarity  

## Models / Techniques Used
- Natural Language Processing (NLP)  
- TF-IDF Vectorization  
- Cosine Similarity Scoring  
- Keyword Extraction  

## Key Results
- Accurately compares resumes with job descriptions  
- Highlights missing skills for better targeting  
- Helps improve ATS resume performance  
- Simplifies the resume screening process
